# Inspiration Vibes Boilerplate
## (Vue2 + Webpack4)
#### pug+stylus

## Installation

1. install node
1. install npm
1. install git
1. Navigate to dir using terminal/command line
1. > git clone https://github.com/kailight/inspiration-vibes-boilerplate.git
1. Navigate to dir/inspiration-vibes-boilerplate
1. > npm run dev
